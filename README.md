# 🏠 Housing Price Prediction

This project focuses on predicting housing prices using machine learning techniques.

---

## 📊 Project Overview
A regression model was developed to estimate house prices based on key features such as:
- Area
- Number of bedrooms
- Number of bathrooms

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

## ⚙️ Methodology
- Data preprocessing and feature selection  
- Train-test split  
- Linear Regression model implementation  
- Model evaluation and prediction  

---

The model predictions were visualized using a scatter plot comparing actual and predicted values.
